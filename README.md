# Introduction 
Windows用のデスクトップアプリケーション。
カレンダー表示ベースでの多人数・複数プロジェクトの作業項目管理ツール。

# Getting Started
1. exeを起動
1. 既に設定データ(xml)がある場合はそれを開く。
1. ない場合は、管理メニューから稼働日とメンバーを編集した後で、作業項目を追加。
1. 詳細はヘルプメニューを参照

# Build and Test
必要な環境はVisual Studio 2019のみ。
